This directory contains the last PK Teach binary file that was built (version 3.3). You will need something like FreeDos or possibly DosBox to be able to run it. It will not run on 64-bit Windows. 

PKTEACH.EXE - The actual PKTeach executible file to run
PKTEACH.EXE.sha256.txt - contains the SHA256 hash (generated with certutil on Windows 7) 
DEMO1.PRG & DEMO2.PRG - The original 2 demo programs supplied with the software for the RS2200 Robot
